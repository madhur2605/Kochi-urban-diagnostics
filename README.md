# 🏙️ Kochi Urban Diagnostics — Index Map Suite

**Urban Immersion Program | TISS × KMC × GIZ | September 2025**

Ward-level spatial index maps developed for Kochi Municipal Corporation (KMC) in collaboration with GIZ.
Each map visualises a distinct dimension of urban health infrastructure and service accessibility across Kochi's wards.

## 🏛️ Context

This work was completed as part of the **Urban Immersion Program (2nd year)** at TISS Mumbai,

Findings and dashboards were presented to KMC and GIZ officials.
